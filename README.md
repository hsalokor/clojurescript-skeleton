# clojurescript-skeleton

Simple clojurescript skeleton.

## Usage

First, run server:

    lein run

On another terminal, set up cljs autocompile

    lein cljsbuild auto

Navigate to URL http://localhost:3000/ and click on the text 'Lol'. Ta-da.
